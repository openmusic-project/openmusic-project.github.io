<div class="headCo">

# <span>Rendering and Playback Control</span>

<div class="headCo_co">

<div>

<div class="infobloc">

<div class="txt">

In OM, most "score" objects can be played via a MIDI synthesizer.
Objects are played :

  - With the space bar, or the `p` / `s` keys from a patch editor.

  - Using the play / stop commands of the palette, or the space bar in
    the score editors

  - When thay are in a maquette, using the maquette playback commands

</div>
