Recording / MIDI In

Some score editors allow the recording of MIDI keybord or other incoming note events.

Use the "RECORD" button of the playing palette: all incoming events will be recorded untill "STOP" is pressed.

The corresponding notes should then appear in the score editor and are stored in the corresponding box.

Input MIDI Port

The input MIDI port number can be changed in the OM MIDI Preferences tab.

The MidiShare Setup panel allows to control the connection of the different output and input port to external devices (e.g. keyboards, etc.).
