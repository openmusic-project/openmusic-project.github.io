<div class="headCo">

# <span>An Alternate Evaluation Tool : the Synthesis Patch</span>

<div class="headCo_co">

<div>

<div class="infobloc">

<div class="txt">

A maquette contains all types of objects : numeric values, chords,
patches... The values of TemporalBoxes within a maquette can be used as
**functional parameters** in a program that will be interpreted
musically. To do so, we need a special type of patch : a synthesis
patch.
