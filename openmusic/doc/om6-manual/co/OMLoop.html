Iterations: OMLoop

A great number of compositional operations, be they basic or more complex, are iterative processes. This is the case, for instance, of transposition, combinatorial manipulations, scale building, and so on. Such processes can be designed in OM via a specific tool : the OMLoop module .

The OMLoop module is available in the OM Kernel / Control menu.
