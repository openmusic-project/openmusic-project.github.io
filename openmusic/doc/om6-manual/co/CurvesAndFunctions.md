<div class="headCo">

# <span>Curves and Functions</span>

<div class="headCo_co">

<div>

<div class="infobloc">

<div class="txt">

The Functions package contains a set of classes and functions designed
to create, edit and manipulate curves, also called **Break Point
Functions** and **Break Point Curves** . These can be accessed via the
`Basic Tools / Functions` menu.

As both classes and functions belong to the Functions package, this
might be slightly confusing. To avoid misunderstandings, we strictly
refer to BPFs, BPCs or curves to mention objects of the package.
