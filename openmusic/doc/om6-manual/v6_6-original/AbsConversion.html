Conversions

Converting an Abstraction into an Internal Abstraction

Procedure

To convert a patch into an internal patch, select the patch box and press a.

Note

The copy of the blue patch that is created is fully autonomous from the master.

Converting an Internal Abstraction into a "Basic" Abstraction

Procedure

Converting a red patch into a blue patch amounts to assigning it a reference in the workspace.

To copy and convert an internal patch into a global – blue – patch, drag and drop the red patch in the workspace.

Modifications : Effects

Once the conversion is done, the abstraction as well as its reference will be equally affected by possible modifications.
