# 1-

![](figures/functions/lisp/minus1.png)

  
  
1-  
  
(lisp module) \-- subtracts 1 from a number  

## Syntax

   **1-**  number  

## Inputs

name| data type(s)| comments  
---|---|---  
  _number_ |  a number|  
  
## Output

output| data type(s)| comments  
---|---|---  
first| a number|  
  
## Description

Subtracts one from  _number_  and returns the result. A useful shortcut.
